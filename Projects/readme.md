# Projects

This folder contains beginner-to-intermediate Machine Learning projects that apply various classification, regression, and deep learning techniques to real-world datasets.

## Project List

### 1. Digit Recognition

Handwritten digit classification using multiple machine learning and deep learning models.

#### Model Performance

| Algorithm           | Accuracy   |
| ------------------- | ---------- |
| SVM                 | 92.80%     |
| Logistic Regression | 92.21%     |
| Random Forest       | 95.86%     |
| CNN                 | **98.99%** |

**Best Performing Model:** Convolutional Neural Network (CNN)

---

### 2. Fake News Detection

A text classification project that identifies fake and real news articles.

**Model Used:**

* Passive Aggressive Classifier

**Techniques:**

* Text preprocessing
* Feature extraction
* News classification

---

### 3. House Price Prediction

End-to-end machine learning application for predicting house prices.

#### Features

* Machine Learning model for price prediction
* Flask API for serving predictions
* Tkinter GUI for user interaction
* Real-time prediction workflow

---

### 4. Sentiment Analysis

Twitter sentiment classification project.

#### Implementation

* Naive Bayes Classifier
* Text-based feature extraction
* CountVectorizer from Scikit-Learn

#### Goal

Classify tweets into sentiment categories based on their textual content.

---

### 5. Spam Email Classification

Email spam detection using Natural Language Processing (NLP) techniques.

#### Implementation

* CountVectorizer
* Multinomial Naive Bayes (MultinomialNB)

#### Dataset

The dataset used for training and evaluation is included in this project folder.
